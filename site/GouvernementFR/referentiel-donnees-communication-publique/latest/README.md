# sig-data-info-gouv-fr

Dépôt de référence des schémas de données et de la documentation des jeux de données ouverts publiés par le **Service d'information du Gouvernement (SIG)** sur [data.gouv.fr](https://www.data.gouv.fr/) et [schema.data.gouv.fr](https://schema.data.gouv.fr/).

Voir le CHANGELOG.md pour l'historique des modifications.

---

## Jeux de données

### 🏷️ Thématiques

Référentiel officiel des thématiques et sous-thématiques de l'action gouvernementale. Structure la navigation du site [info.gouv.fr](https://www.info.gouv.fr/) et le classement de tous les contenus publiés.

- Fichier de données : `thematiques.json` *(mis à jour automatiquement par [info.gouv.fr](http://info.gouv.fr/))*
- Documentation : `thematiques/README.md`

---

### 🏛️ Composition du Gouvernement

Composition officielle du Gouvernement en exercice : membres, fonctions, rattachements ministériels, tels que publiés au Journal officiel.

- Fichier de données : `composition-gouvernement.json` *(mis à jour automatiquement par [info.gouv.fr](http://info.gouv.fr/))*
- Fichier de données : `ministeres.json` *(mis à jour automatiquement par [info.gouv.fr](http://info.gouv.fr/))*
- Documentation : `composition-gouvernement/README.md`

---

### 🎯 Missions essentielles de l'État

Les six missions essentielles de l'État, telles que définies par le SIG. Chaque mission se décline en objets de communication interministérielle.

- Fichier de données : `missions-essentielles.json`
- Documentation : `missions-essentielles/README.md`

---

### 📰 Communiqués de presse

Schéma de données officiel des communiqués de presse du Gouvernement. Harmonise les pratiques de publication entre les ministères et leurs services.

- Fichier de données : `communiques.jsonl`
- Schéma JSON : `schema.json` *(publié sur [schema.data.gouv.fr](https://schema.data.gouv.fr/sig/schema-communiques-presse-gouvernement/schema.json))*
- Documentation données : `communiques-de-presse/README-donnees.md`
- Documentation schéma : `communiques-de-presse/README-schema.md`
